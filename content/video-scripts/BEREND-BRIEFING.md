# Demo Video Briefing für Berend

> **Ziel:** 15-Sekunden Video das konvertiert. Kein Stress, alles vorbereitet.

---

## VOR DEM RECORDING (5 Min Setup)

### 1. Kleidung
- **JA:** Einfarbiges T-Shirt oder Hemd (dunkelblau, schwarz, grau)
- **NEIN:** Streifen, Logos, grelle Farben, Weiß (überstrahlt)
- **Warum:** Facecam ist klein - einfache Farben lenken nicht ab

### 2. Hintergrund
- **IDEAL:** Neutrale Wand, Bücherregal, Pflanzen
- **VERMEIDEN:** Unordnung, Fenster mit Gegenlicht, Bett
- **QUICK FIX:** Blur-Effekt in OBS (Filter → "Background Removal")

### 3. Lighting
- **REGEL:** Licht VOR dir, nicht hinter dir
- **QUICK:** Schreibtischlampe auf Gesicht richten
- **BESSER:** Fenster vor dir (Tageslicht)
- **SCHLECHT:** Fenster hinter dir = Silhouette

### 4. Facecam Position
- **WO:** Unten links oder unten rechts im Video
- **GRÖSSE:** Ca. 20% der Bildschirmbreite
- **OBS:** Sources → Video Capture → Resize + Position

## WÄHREND DEM RECORDING

### Was du SAGST (optional - kann auch stumm sein):

**Version A: Mit Voice (15 Sek)**
```
[0:00] "Schau dir diese Review an..."
[0:03] "Das zerstört gerade dein Business."
[0:05] *klick auf Extension*
[0:07] "Ein Klick..."
[0:10] *Response erscheint*
[0:11] "...und fertig."
[0:13] "tryreviewresponder.com"
```

**Version B: Stumm (Text-Overlays machen den Job)**
- Du zeigst nur mit der Maus
- Text-Overlays sind schon im Script
- Einfacher, weniger Fehler

### Körpersprache
- **Blick:** Auf den Bildschirm (nicht in Kamera)
- **Ausdruck:** Leicht genervt bei negativer Review → überrascht bei AI Response
- **Hände:** Sichtbar bei Facecam? → Ruhig halten

## OBS SETUP (einmalig)

### Facecam hinzufügen:
1. Sources → + → Video Capture Device
2. Deine Webcam auswählen
3. Resize auf ~300x200 Pixel
4. Position: Unten links
5. Optional: Filter → Background Removal

### Scene Setup:
```
┌────────────────────────────────────┐
│                                    │
│     [Chrome Window Capture]        │
│                                    │
│                                    │
│ ┌──────┐                           │

│ └──────┘                           │
└────────────────────────────────────┘
```

## FALLS ES NICHT KLAPPT

| Problem | Lösung |
|---------|--------|
| Facecam zu dunkel | Mehr Licht von vorne |
| Hintergrund hässlich | OBS Background Blur |
| Stimme klingt komisch | Stumme Version machen |
| Demo zu lang | Schneller klicken, weniger reden |
| Nervös | 3x durchatmen, es sind nur 15 Sekunden |
