# Skill Updates - Self-Learning System

> Jeder Night-Burst Agent kann hier Learnings dokumentieren,
> die DIREKT in die Skill-Files geschrieben werden sollen.

---

## Wie es funktioniert:

1. Agent entdeckt Problem oder Learning
2. Agent schreibt hier die Änderung auf
3. Agent FÜHRT die Änderung durch (Edit des Skill-Files)
4. Nächster Claude profitiert sofort
