# Chrome Web Store Screenshots

Screenshots für die Chrome Web Store Submission müssen hier abgelegt werden.

## Benötigte Screenshots (1280x800 px)

| Dateiname | Beschreibung | Status |
|-----------|--------------|--------|
| `screenshot-1-overview.png` | Google Maps mit ReviewResponder Button | TODO |
| `screenshot-2-popup.png` | Extension Popup mit generierter Response | TODO |
| `screenshot-3-tones.png` | Tone-Auswahl Dropdown | TODO |
| `screenshot-4-dashboard.png` | Web Dashboard mit History/Analytics | TODO |
| `screenshot-5-result.png` | Kopierte Response, Success Message | TODO |

## Anforderungen

- **Größe:** 1280x800 px (oder 640x400 px)
- **Format:** PNG oder JPEG
- **Mindestens:** 1 Screenshot erforderlich
- **Empfohlen:** 3-5 Screenshots

## Anleitung zum Erstellen

1. Öffne Chrome im Fenster mit 1280x800 px Größe
2. Navigiere zu Google Maps und suche ein Business
3. Zeige die Extension-Funktionalität
4. Screenshot mit Snipping Tool / Screenshot-Tool
5. Speichere als PNG in diesem Ordner

## Tipps

- Zeige die wichtigsten Features
- Verwende echte (nicht sensible) Daten
- Helle, gut lesbare Screenshots
- Zeige verschiedene Tones/Optionen
