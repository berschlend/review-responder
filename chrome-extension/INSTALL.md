# ReviewResponder Chrome Extension - Installation Guide

## Quick Installation (2 minutes)

### Step 1: Create Icons
1. Open `icons/create-icons.html` in Chrome
2. Click "Download" for each icon (icon16.png, icon48.png, icon128.png)
3. Move downloaded files to the `icons` folder

### Step 2: Load Extension in Chrome
1. Open Chrome
2. Go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top-right corner)
4. Click "Load unpacked"
5. Select the `chrome-extension` folder

### Step 3: Login
1. Click the ReviewResponder icon in your Chrome toolbar
2. Login with your ReviewResponder account
3. If you don't have an account, click "Sign up free"

## How to Use

### Option A: Use the Popup
1. Click the ReviewResponder icon in Chrome
2. Paste any review into the text field
3. Select your preferred tone
4. Click "Generate Response"
5. Click "Copy to Clipboard"

### Option B: Use on Google Maps (Automatic)
1. Go to Google Maps (google.com/maps)
2. Search for any business
3. Look at their reviews
4. You'll see a "Generate Response" button next to each review
5. Click it to generate a response instantly

## Troubleshooting

### Extension not working?
- Make sure you're logged in (click the extension icon)
- Check that you have responses remaining in your plan
- Try refreshing the page

### Buttons not appearing on Google Maps?
- Some review layouts may not be detected automatically
- Use the popup method instead (Option A)

### Can't login?
- Make sure you have an account at https://review-responder-frontend.onrender.com
- Check your email and password

## Updating the Extension
When there's an update:
1. Download the new version
2. Go to `chrome://extensions/`
3. Click the refresh icon on the ReviewResponder extension
