ReviewResponder Chrome Extension - Installation Guide
=====================================================

STEP 1: Create the icons
------------------------
1. Open "create-icons.html" in Chrome
2. Click "Download" for each icon (icon16.png, icon48.png, icon128.png)
3. Save them in the same folder as this file

STEP 2: Load in Chrome
----------------------
1. Open Chrome
2. Go to: chrome://extensions/
3. Enable "Developer mode" (toggle in top-right corner)
4. Click "Load unpacked"
5. Select THIS FOLDER (the one containing manifest.json)

STEP 3: Login & Use
-------------------
1. Click the ReviewResponder icon in Chrome toolbar
2. Login with your account (create one at https://review-responder-frontend.onrender.com)
3. Paste any review and generate responses!
